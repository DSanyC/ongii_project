# Ongii (온기) - 제품 개요

## 1. 프로젝트 배경

가족 카카오톡 채팅방은 소통은 활발하지만,
- 대화 주제가 뒤섞이고
- 사진, 일정, 공지가 한 흐름에 묻히며
- 과거 기록을 찾기 어렵다.

Ongii는 가족의 시간을 달력 중심으로 정리하여
일정과 추억을 구조적으로 보관하는 가족 전용 SNS이다.

---

## 2. 핵심 철학

카카오톡 = 대화 중심  
Ongii = 기록 중심

---

## 3. 핵심 사용자 전략

모두가 활발히 사용하는 앱이 아니라,
"잘 사용하는 1~2명이 중심이 되는 구조"

예시:
- 아버지가 일정과 기록을 정리
- 나머지 가족은 열람 위주

---

## 4. 앱 구조

하단 탭 4개:

[ 달력 ] [ 기록 ] [ 채팅 ] [ 마이 ]

- 달력 = 중심 기능
- 기록 = 사진/글 정리 공간
- 채팅 = 빠른 소통
- 마이 = 설정 및 알림 관리
